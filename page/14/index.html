vue的组件

发表于 2017-05-17 | 分类于 js |

TL;DR

  • 全局注册组件：Vue.component('my-component', {template: '<div>A custom component!</div>'})
  • 局部注册组件：new Vue({components: {'my-component': {template: '<div>A custom component!</div>'} } })
  • 组件里面的数据data必须是函数

  • Props 允许外部环境传递数据给组件，父组件通过自定义属性传递数据，子组件通过props接受数据(属性名)

    如果message是变动的也ok，:message="variable"
    如果子组件需要改变父组件传过来的数据，需要自己自定义变量接收，或者用computed
    注意，如果数据是引用类型，需要深复制。

  • props验证，父组件的属性可以定义规则，包括类型等等

  • Events 允许组件触发外部环境的副作用，父组件使用 $on(eventName) 监听事件，子组件使用 $emit(eventName) 触发事件

  • Slots 允许外部环境将额外的内容组合在组件中。

vue的计算属性

发表于 2017-05-17 | 分类于 js |

TL;DR

  • 插值的四种用法：
    • 文本插值
    • html插值v-html="rawHtml"
    • 属性插值v-bind:id="someId"
    • js配合Mustache插值17
  • 指令（Directives）是带有 v- 前缀的特殊属性职责就是当其表达式的值改变时相应地将某些行为应用到 DOM 上。
  • 一些指令能接受一个“参数”，在指令后以冒号指明。v-bind:href="url
  • 修饰符（Modifiers）用于指出一个指令应该以特殊方式绑定。v-on:submit.prevent="onSubmit"，一般用于v-on,v-model
  • 过滤器可以用在两个地方：mustache 插值和 v-bind 表达式。
  • Vue.js 允许你自定义过滤器，可被用作一些常见的文本格式化。
  • 缩写 :
    • v-bind:href=”url -> :href=”url”
    • v-on:click=”doSomething” -> @click=”doSomething”

vue的表单控件绑定

发表于 2017-05-17 | 分类于 js |

TL;DR

  • v-model在表单控件元素上创建双向数据绑定。v-model="message"双向的意思是，input这边的输入值变化会让message变化，而message变化会让input的值变化。
  • v-model在文本框内就是文本框的值
  • v-model在复选框里，默认是布尔值，当有多个复选框时，要设置为数组，选中某项则数组中就会出现此项的值
  • v-model在选择列表里，放在select标签上，等于选中的option的值
  • v-model一般是input事件，修饰符lazy，可以将其变成change事件
  • 修饰符三种：lazy、number、trim（基本都是文本框）

vue的事件处理器

发表于 2017-05-17 | 分类于 js |

TL;DR

  • 直接写js代码：v-on:click=”counter += 1”
  • 方法事件处理器：v-on:click=”greet”
  • 内联处理器，需要event的时候必须传$event：v-on:click=”greet(param,$event)”，需要注意的是内联处理器必须是appvue实例里声明的方法
  • 事件修饰符：v-on:click.stop.prevent.once.self.capture=”greet”
  • 按键修饰符：v-on:keyup.enter.left.right=”greet”
  • 事件处理器的好处：一目了然，不需要找选择器，不需要时会自动销毁
