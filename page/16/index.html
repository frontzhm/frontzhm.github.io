7.8.9月辅食的制作

发表于 2018-10-12 | 分类于 baby |

7月辅食制作公式

辅食泥的做法

辅食泥制作步骤: 

原料烹熟 =>    
    1. 勺子背压泥  
    1. 研磨碗捣鼓成泥  
    1. 压蒜器挤成泥状,特别适合叶类肉类

辅食泥做好后可以放冰格存起来,要的时候拿出来加热即可.
下面的面和粥就是,主食煮好之后,从冰箱拿出泥搅拌即可,很方便.

颗粒面的做法

颗粒面的制作步骤: 

面煮熟 => 加入配菜搅拌

原料: 面 + 液体 + 配料 

1. 颗粒面(GB10769 含铁)  
1. 液体(水,牛奶,高汤)
1. 配料(菜 蛋 肉 鱼 虾 坚果粉末))

比如: 南瓜青菜虾泥颗粒面的制作

1. 放颗粒面
2. 用水煮熟
3. 加入南瓜青菜虾泥搅拌

比如: 红薯油菜三文鱼鸡汤面

1. 放颗粒面
2. 放鸡汤煮熟
3. 加入红薯油菜三文鱼搅拌

十倍粥的做法

十倍粥: 10g粥 + 100g水 (水是粥的十倍)

制作步骤: 

粥煮熟 => 加入配菜

原料: 主食 + 液体 + 配料 

1. 粮食主食: 大米,小米,玉米,高粱米,红豆,绿豆,红薯
2. 液体: 水 奶 高汤
3. 配料: 菜 肉 蛋 水果 果干

比如: 西蓝花三文鱼大小米粥 

1. 大米小米加水煮熟
2. 然后加入西蓝花三文鱼煮一煮 也可以搅拌

比如: 苹果南瓜蛋黄米粥(生病的时候宝宝也会喜欢吃 甜甜的细腻的) 

1. 大米加水和一点奶粉煮熟
2. 然后苹果南瓜蛋黄煮一煮 或者搅拌

常用的配料: 土豆胡萝卜泥 青菜泥 火龙果 三文鱼 紫甘蓝泥 土豆鸡肉泥 肝粉

6-7月辅食和奶的安排

什么时候吃辅食

和奶一起喝
也可以和家人一起进餐,这样做饭时间短.

比如:
7.30喝奶
11.30辅食+喝奶
3.30辅食+喝奶
6喝奶
8喝奶

先吃辅食还是先喝奶

原则: 
1. 6-7月的辅食原则是不影响喝奶量,只是简单的尝味道以及补充部分营养,所以这阶段的辅食是很少量的
2. 辅食之后立即喝奶,因为如果分开,进食就会变成多次,会让宝宝的饥饿感不明显,这样降低宝宝对食物的兴趣,会觉得吃不吃无所谓,长期以往是不利于宝宝成长的

遇到问题的话参照的原则:
1. 如果吃辅食的时候情绪很差不愿意吃,可以先吃点点奶缓解宝宝的情绪(这里的量不多 仅仅为了缓解情绪),然后吃辅食,然后在吃奶吃到饱
2. 如果只想吃辅食不喝奶的话,那就调换顺序.
大原则就是: 6-7月龄的必须保证奶量.

辅食和奶需要多少

辅食一天1-2次即可.


进餐安排图片在微信

喝奶变少

6-7月龄如果因为辅食量影响奶量,就会影响宝宝的生长发育,所以奶量必须保证.
喝奶变少的话 就必须减少辅食量

8月龄辅食制作公式

8月龄食物的制作 - 辅食碎末(促进宝宝咀嚼)

碎末制作方法: 

1. 厨房搅馅器搅碎 (生的肉 菜 胡萝卜等) 再蒸或者煮熟
2. 弄熟之后 勺子戳碎 (土豆 胡萝卜 红薯类)
3. 菜刀剁碎
4. 辅食剪剪碎(面条 肉)

细面

汤面的一般步骤: 

面煮熟 - 加入配菜 - 剪成颗粒状

炒面的一般步骤: 

面煮熟 - 配菜炒熟 - 加入面条炒匀 - 剪成颗粒状

原料: 面(GB10769) + 液体(水 奶 汤) + 配料())

比如: 胡萝卜生菜虾肉面 

面(细面) + 液体(虾汤) + 配料(胡萝卜 生菜 虾肉))

比如: 番茄油麦菜肉末面 

面(细面) + 液体(水) + 配料(油麦菜 番茄 肉末)

面疙瘩

面疙瘩的三种制作方法: 

1. 漏勺法: 搅好面糊(放进有孔的漏勺里,用东西压下,这样面糊就顺着孔进锅里) - 煮熟疙瘩 - 加入配菜
1. 挖块法: 搅好面糊(面糊就是面和水搅拌,稍微干燥些,用小勺子挖进锅里) - 煮熟疙瘩 - 加入配菜
1. 滴水法: 滴水(打开水龙头 滴水到面粉里 一边滴一边摇匀)摇匀成疙瘩 - 煮熟疙瘩 - 加入配菜

原料: 粉 - 调糊 - 配料

1. 粉(面粉 可以加点全麦粉 玉米面 米粉 坚果粉之类)
2. 调糊 (水 奶 高汤 鸡蛋液 辅食汁[汁可以上色]  )
3. 配料(一堆)

比如: 南瓜泥青菜面疙瘩

原料: 粉(面粉 玉米面) + 调糊(南瓜泥 水) + 配料(青菜)

比如: 香菇巴沙鱼西蓝花鸡蛋面疙瘩

原料: 粉(面粉 婴儿米粉) + 调糊(紫甘蓝泥 水) + 配料(香菇巴沙鱼西蓝花鸡蛋)

肉丸

肉丸做好之后放入冰箱冷冻起来,需要的话拿出来蒸熟/煮熟即可.

一般步骤: 

肉打成泥 - 加入配料 - 塑性(圆形 片状 肉肠状) - 蒸熟/煮熟

其中圆形: 手搓即可
片状: 放在方形的容器蒸熟 拿出来切片即可
肉肠状: 肉肠状模具 蒸熟

原料: 肉 + 调糊 + 配料

1. 肉像(猪肉 牛肉 鸡肉  鸡肉 虾肉等)打成泥
2. 调糊(淀粉[玉米淀粉 马铃薯淀粉] 蛋液 奶 水 ),肉的粘性差加入前面那些增强粘性,如果肉的水分多就不用加水了
3. 配料(一点点即可 青菜泥 南瓜泥 菜的碎末 坚果的粉末等)

比如: 小白菜猪肉丸子 

原料: 肉(猪肉) + 调糊(淀粉) + 配料(小白菜 虾皮粉)

比如: 胡萝卜牛肉丸 

原料: 肉(牛肉) + 调糊(蛋清) + 配料(胡萝卜 土豆 香菇)

饼

一般步骤: 调面糊 - 烙熟

松饼

一般步骤: 

打发蛋白 - 混合面糊 - 烙熟面糊

1. 蛋清用电动打蛋器打成糊糊状
2. 蛋黄和面在一起
3. 激活酵母(放入温水) - 混合面糊 - 等待发酵 (30min) - 烙熟

原料: 粉 - 调糊 - 配料

1. 粉(面粉 可以加点全麦粉 玉米面 米粉 坚果粉之类)
2. 调糊 (水 奶 高汤 鸡蛋液 辅食汁[汁可以上色]  )
3. 配料(一堆)

比如: 菠菜胡萝卜饼 

原料: 粉(面粉) + 调糊(水) + 配料(菠菜 胡萝卜 甜椒))

比如:香葱酸奶玉米鸡蛋饼 

原料: 粉(面粉 玉米面) + 调糊(鸡蛋 酸奶) + 配料(香葱))

馒头/发糕

发糕步骤: 

调面糊 - 发酵 - 蒸熟

馒头步骤: 

揉面团 - 发酵 - 再揉下排气 - 二次发酵 - 蒸熟

原料: 

粉(面粉 可以加点全麦粉 玉米面 米粉 坚果粉之类) - 调糊 (安琪干酵母 奶 水 ) - 配料(紫薯泥 鸡蛋 南瓜泥 青菜泥)

比如: 馒头

原料: 粉(面粉 奶粉) - 调糊 (安琪干酵母 水 ) - 配料(胡萝卜汁 菠菜汁))

比如: 发糕 

原料: 粉(面粉) - 调糊 (安琪干酵母 水 ) - 配料(紫薯泥 山药泥))

蛋羹

一般步骤: 

搅拌蛋液 - 加入配料 - 蒸熟

原料: 

蛋(鸡蛋 鸭蛋 鹅蛋 鹌鹑蛋) + 液体(奶 温水 高汤) + 配料(鱼 虾 菜 )

小窍门: 温水调蛋液 蛋会很滑嫩

比如: 秋葵蒸蛋羹 (秋葵切片以后像小花)) 

原料:蛋(鸡蛋) + 液体(温水) + 配料(秋葵)

比如:香菇菠菜虾仁蛋羹 

原料:蛋(鸡蛋) + 液体(温水) + 配料(虾 香菇 菠菜)

宝宝学会吃饭

发表于 2018-10-12 | 分类于 baby |

进食能力的发展[11-1]



进食能力包括两方面: 

1. 自主进食能力(食物从面前到嘴里:抓 握 捏 舀 夹 递)
1. 咀嚼吞咽的能力(食物从嘴里到肚子里: 送 吞 嚼 拌 吐 连)

为什么要让宝宝自己吃[11-2]

  1. 手眼口协调,精细动作发育
  2. 增强自信,独立,责任感
  3. 增强进食兴趣,减少拒食
  4. 共同进餐,解放家长的时间
  5. 权威机构的建议:

开始学习自己吃饭的时间[11-3 10min左右]

7.5个月左右的时间全手抓握,10个月左右的时间拇指抓握

看信号,任意信号就是想自己吃饭的意愿: 

宝宝抢勺子,抢碗 => 试试
宝宝抓面前的食物 => 香蕉 胡萝卜
宝宝闹(不乐意吃 咬勺子))

手指食物[11-4]

手指食物的形状 

1. 初始阶段,只知道全手握:长条状,不小于6厘米,大人小拇指粗
2. 后续阶段,可以打开手吃,可以换手: 随便什么形状,她可以抓起来吃就行

增加食物表面粗糙: 

1. 外面裹米粉,如香蕉,牛油果等等
1. 波浪刀
1. 球状食物一定切开

初始手指食物的硬度: 软!软!软!(手一捏就烂了,苹果不行的,如黄瓜 西蓝花 胡萝卜 土豆)

后续手指食物: 学会咀嚼之后可以宽松点,因为会咀嚼了嘛,可以一锅煮

初始食物的要求: 能抓,能捏烂


90种手指食物汇总

用勺子吃饭[11-5]

用手: 6-18个月
用勺子:10-18个月
练习次数是关键: 

第一阶段:好孩子弯头勺(10-14m)
第二阶段:oxo金属勺(12-24m)
第三阶段:爱迪生勺子(24-36m)

碗: 深/稳[防滑垫 或者 吸盘]

开始的时候用增加成功率的食物[能180度翻转,粘性强]: 稠厚的米粉 食物泥 软焖饭 稠密的粥 酸奶

开始的时候,舀好食物放着.因为这步比较难.容易挫伤宝宝.然后宝宝拿着有食物的勺子送到嘴里就ok了.
观察,如果哪步对宝宝有困难就帮助她.

为什么不自己吃呢[11-6]

意愿 + 能力
能力的提高: 

1. 递送东西(抓握东西 传递东西 把那个递给妈妈)
2. 打开手心(手心里放个小泡芙 哇 手心里有什么)
3. 舀取搬运(比如一个放满玉米粒,一个空着,让宝宝练习搬运)

意愿: 

1. 尽量随着宝宝
1. 喝汤喽,擦一擦绘本
1. 相互喂食(为妈妈吃)
1. 让宝宝多观察大人吃饭

一千个喂饭的理由[11-7]

他还是个孩子啊,[给他机会试试]
有一种不行叫我觉得你不行[权威机构]
干呕好可怕,不敢给了[干呕是正常的保护机制 避免食物引起窒息]
需要注意: 

1. 宝宝在餐椅里面坐直
2. 去掉干扰源,专心进餐
3. 给合适的条状,不要太小
4. 吃东西的时候,家长必须看护

噎住的话有视频[多次看]

快手早餐的例子

发表于 2018-10-12 | 分类于 baby |

快手早餐合集例子[8-8]

胡萝卜土豆香菇丝（提前泡发）

  1. 提前泡发干贝和香菇
  2. 胡萝卜，土豆，刀豆切成丝，时间长就泡在水里
    上面两步可以提前做，放入冰箱里
  3. 泡好的香菇切成丝，干贝捏碎
  4. 倒油，加上所有素材，炒一会，倒入香菇水闷会，然后放盐出锅

西蓝花胡萝卜

  1. 西蓝花切小朵，胡萝卜切小块，锅里煮一会，颜色变深捞起来
    上面可以做好，放冰箱
  2. 放油，放素材，炒一会，加水焖一会，然后放盐出锅

鸡蛋牛奶饼（有点麻烦）

1.磕个鸡蛋，蛋清蛋黄分开
2.蛋黄里加10g面粉，一点糖，一点油拌匀
上面的做好后，可以放进冰箱里，顺手准备好 煎锅 打蛋器

  1. 向蛋黄糊里加一点牛奶拌匀
  2. 打发蛋白到提起来有小弯钩
  3. 蛋黄糊倒到蛋清里搅拌
  4. 煎锅，小火，面糊凝固后翻面继续，熟了就可以了

饭饼

  1. 生菜 小白菜 口蘑(白色的小蘑菇)洗干净,放锅里煮熟
  2. 捞出来切成菜碎，和饭拌匀
    上面可以放入冰箱
  3. 在上面的饭里，磕一个鸡蛋，加点香菇粉(今良)拌成糊状
  4. 小火，倒入米饭糊，煎到两面发黄就可以了

燕麦糊

  1. 燕麦片，水，红枣，少量杏仁，预约煮粥
    上面可以提前约
  2. 用料理棒打成糊

菠菜虾仁蛋饼（有点难度）

  1. 菠菜焯水2分钟，剥虾仁，
  2. 菜和虾仁剁碎
    上面可以放入冰箱
  3. 磕一个鸡蛋，加点油，加点香菇粉拌匀

  4. 倒入不粘锅晃动，摊开之后开小火，未凝固的时候放入菜和虾仁，
    (如果凝固了再卷,蛋卷会散开),立刻卷起(锅里放油会让蛋皮硬点更容易卷起),用铲子一边卷那种,卷完之后推到中间,在上方轻按,按不出蛋液,表面变黄就熟了

  5. 蛋卷切块

馒头片（）

  1. 冷冻馒头 蛋液
  2. 冷冻馒头蒸熟后放着不用管了（九州丰园儿童馒头 12元）
  3. 馒头切片，鸡蛋打散，馒头两面沾了蛋液之后放进不粘锅，多翻几次

1. 杂粮洗净（大米燕麦小米黑米等），放去核的红枣

1. 准备擦丝板（多功能切菜器 30元）和不粘锅，sleep

1. 煎鱼的时候切好胡萝卜细丝，秋葵切片（横着切 有小花 好看），黄瓜切丝
1. 鱼两面发黄就熟了，盛出来，锅里的油直接放入蔬菜翻炒
1. 加少量水和盐，焖一会（宝宝不加盐）
1. 各种装盘，玉米切小段

例子16： 

1. 调味牛排提前解冻（小宝宝可换成鸡胸肉）
1. 意面泡水
1. 准备好汤锅，不粘锅睡觉



早上：
1. 煮一锅水烧开，加入油和盐（小宝宝不加盐）
1. 切几片土豆先煮，切点胡萝卜西蓝花，一起煮
1. 泡好的意面放进去，一家人要整整齐齐
1. 锅里刷油放入牛排，小火，两面变色后取出来切小，重新放进锅里煎熟
1. 西蓝花煮熟捞出，牛排捞出
1. 锅里倒入一些番茄酱（冠农番茄丁 8元），倒入煮好的意面，拌匀即可
1. 胡萝卜土豆捞出来戳碎（小宝宝碾成泥），可以搞一个小团子装盘，再倒牛奶装盘

例子17： 

1. 杂粮洗净加水（黑米 燕麦 糙米），加入去核红枣
1. 另一个炖盅放几片胡萝卜，不加水，预约煮粥

例子18：

例子19：

例子21：
