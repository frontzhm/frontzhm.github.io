7.8.9月辅食的制作

发表于 2018-10-12 | 分类于 baby |

7月辅食制作公式

辅食泥的做法

辅食泥制作步骤: 

原料烹熟 =>    
    1. 勺子背压泥  
    1. 研磨碗捣鼓成泥  
    1. 压蒜器挤成泥状,特别适合叶类肉类

辅食泥做好后可以放冰格存起来,要的时候拿出来加热即可.
下面的面和粥就是,主食煮好之后,从冰箱拿出泥搅拌即可,很方便.

颗粒面的做法

颗粒面的制作步骤: 

面煮熟 => 加入配菜搅拌

原料: 面 + 液体 + 配料 

1. 颗粒面(GB10769 含铁)  
1. 液体(水,牛奶,高汤)
1. 配料(菜 蛋 肉 鱼 虾 坚果粉末))

比如: 南瓜青菜虾泥颗粒面的制作

1. 放颗粒面
2. 用水煮熟
3. 加入南瓜青菜虾泥搅拌

比如: 红薯油菜三文鱼鸡汤面

1. 放颗粒面
2. 放鸡汤煮熟
3. 加入红薯油菜三文鱼搅拌

十倍粥的做法

十倍粥: 10g粥 + 100g水 (水是粥的十倍)

制作步骤: 

粥煮熟 => 加入配菜

原料: 主食 + 液体 + 配料 

1. 粮食主食: 大米,小米,玉米,高粱米,红豆,绿豆,红薯
2. 液体: 水 奶 高汤
3. 配料: 菜 肉 蛋 水果 果干

比如: 西蓝花三文鱼大小米粥 

1. 大米小米加水煮熟
2. 然后加入西蓝花三文鱼煮一煮 也可以搅拌

比如: 苹果南瓜蛋黄米粥(生病的时候宝宝也会喜欢吃 甜甜的细腻的) 

1. 大米加水和一点奶粉煮熟
2. 然后苹果南瓜蛋黄煮一煮 或者搅拌

常用的配料: 土豆胡萝卜泥 青菜泥 火龙果 三文鱼 紫甘蓝泥 土豆鸡肉泥 肝粉

6-7月辅食和奶的安排

什么时候吃辅食

和奶一起喝
也可以和家人一起进餐,这样做饭时间短.

比如:
7.30喝奶
11.30辅食+喝奶
3.30辅食+喝奶
6喝奶
8喝奶

先吃辅食还是先喝奶

原则: 
1. 6-7月的辅食原则是不影响喝奶量,只是简单的尝味道以及补充部分营养,所以这阶段的辅食是很少量的
2. 辅食之后立即喝奶,因为如果分开,进食就会变成多次,会让宝宝的饥饿感不明显,这样降低宝宝对食物的兴趣,会觉得吃不吃无所谓,长期以往是不利于宝宝成长的

遇到问题的话参照的原则:
1. 如果吃辅食的时候情绪很差不愿意吃,可以先吃点点奶缓解宝宝的情绪(这里的量不多 仅仅为了缓解情绪),然后吃辅食,然后在吃奶吃到饱
2. 如果只想吃辅食不喝奶的话,那就调换顺序.
大原则就是: 6-7月龄的必须保证奶量.

辅食和奶需要多少

辅食一天1-2次即可.


进餐安排图片在微信

喝奶变少

6-7月龄如果因为辅食量影响奶量,就会影响宝宝的生长发育,所以奶量必须保证.
喝奶变少的话 就必须减少辅食量

8月龄辅食制作公式

8月龄食物的制作 - 辅食碎末(促进宝宝咀嚼)

碎末制作方法: 

1. 厨房搅馅器搅碎 (生的肉 菜 胡萝卜等) 再蒸或者煮熟
2. 弄熟之后 勺子戳碎 (土豆 胡萝卜 红薯类)
3. 菜刀剁碎
4. 辅食剪剪碎(面条 肉)

细面

汤面的一般步骤: 

面煮熟 - 加入配菜 - 剪成颗粒状

炒面的一般步骤: 

面煮熟 - 配菜炒熟 - 加入面条炒匀 - 剪成颗粒状

原料: 面(GB10769) + 液体(水 奶 汤) + 配料())

比如: 胡萝卜生菜虾肉面 

面(细面) + 液体(虾汤) + 配料(胡萝卜 生菜 虾肉))

比如: 番茄油麦菜肉末面 

面(细面) + 液体(水) + 配料(油麦菜 番茄 肉末)

面疙瘩

面疙瘩的三种制作方法: 

1. 漏勺法: 搅好面糊(放进有孔的漏勺里,用东西压下,这样面糊就顺着孔进锅里) - 煮熟疙瘩 - 加入配菜
1. 挖块法: 搅好面糊(面糊就是面和水搅拌,稍微干燥些,用小勺子挖进锅里) - 煮熟疙瘩 - 加入配菜
1. 滴水法: 滴水(打开水龙头 滴水到面粉里 一边滴一边摇匀)摇匀成疙瘩 - 煮熟疙瘩 - 加入配菜

原料: 粉 - 调糊 - 配料

1. 粉(面粉 可以加点全麦粉 玉米面 米粉 坚果粉之类)
2. 调糊 (水 奶 高汤 鸡蛋液 辅食汁[汁可以上色]  )
3. 配料(一堆)

比如: 南瓜泥青菜面疙瘩

原料: 粉(面粉 玉米面) + 调糊(南瓜泥 水) + 配料(青菜)

比如: 香菇巴沙鱼西蓝花鸡蛋面疙瘩

原料: 粉(面粉 婴儿米粉) + 调糊(紫甘蓝泥 水) + 配料(香菇巴沙鱼西蓝花鸡蛋)

肉丸

肉丸做好之后放入冰箱冷冻起来,需要的话拿出来蒸熟/煮熟即可.

一般步骤: 

肉打成泥 - 加入配料 - 塑性(圆形 片状 肉肠状) - 蒸熟/煮熟

其中圆形: 手搓即可
片状: 放在方形的容器蒸熟 拿出来切片即可
肉肠状: 肉肠状模具 蒸熟

原料: 肉 + 调糊 + 配料

1. 肉像(猪肉 牛肉 鸡肉  鸡肉 虾肉等)打成泥
2. 调糊(淀粉[玉米淀粉 马铃薯淀粉] 蛋液 奶 水 ),肉的粘性差加入前面那些增强粘性,如果肉的水分多就不用加水了
3. 配料(一点点即可 青菜泥 南瓜泥 菜的碎末 坚果的粉末等)

比如: 小白菜猪肉丸子 

原料: 肉(猪肉) + 调糊(淀粉) + 配料(小白菜 虾皮粉)

比如: 胡萝卜牛肉丸 

原料: 肉(牛肉) + 调糊(蛋清) + 配料(胡萝卜 土豆 香菇)

饼

一般步骤: 调面糊 - 烙熟

松饼

一般步骤: 

打发蛋白 - 混合面糊 - 烙熟面糊

1. 蛋清用电动打蛋器打成糊糊状
2. 蛋黄和面在一起
3. 激活酵母(放入温水) - 混合面糊 - 等待发酵 (30min) - 烙熟

原料: 粉 - 调糊 - 配料

1. 粉(面粉 可以加点全麦粉 玉米面 米粉 坚果粉之类)
2. 调糊 (水 奶 高汤 鸡蛋液 辅食汁[汁可以上色]  )
3. 配料(一堆)

比如: 菠菜胡萝卜饼 

原料: 粉(面粉) + 调糊(水) + 配料(菠菜 胡萝卜 甜椒))

比如:香葱酸奶玉米鸡蛋饼 

原料: 粉(面粉 玉米面) + 调糊(鸡蛋 酸奶) + 配料(香葱))

馒头/发糕

发糕步骤: 

调面糊 - 发酵 - 蒸熟

馒头步骤: 

揉面团 - 发酵 - 再揉下排气 - 二次发酵 - 蒸熟

原料: 

粉(面粉 可以加点全麦粉 玉米面 米粉 坚果粉之类) - 调糊 (安琪干酵母 奶 水 ) - 配料(紫薯泥 鸡蛋 南瓜泥 青菜泥)

比如: 馒头

原料: 粉(面粉 奶粉) - 调糊 (安琪干酵母 水 ) - 配料(胡萝卜汁 菠菜汁))

比如: 发糕 

原料: 粉(面粉) - 调糊 (安琪干酵母 水 ) - 配料(紫薯泥 山药泥))

蛋羹

一般步骤: 

搅拌蛋液 - 加入配料 - 蒸熟

原料: 

蛋(鸡蛋 鸭蛋 鹅蛋 鹌鹑蛋) + 液体(奶 温水 高汤) + 配料(鱼 虾 菜 )

小窍门: 温水调蛋液 蛋会很滑嫩

比如: 秋葵蒸蛋羹 (秋葵切片以后像小花)) 

原料:蛋(鸡蛋) + 液体(温水) + 配料(秋葵)

比如:香菇菠菜虾仁蛋羹 

原料:蛋(鸡蛋) + 液体(温水) + 配料(虾 香菇 菠菜)

oss的好处

刚接触的oss，很多东西不清楚，这篇仅为了记录以及加强理解，如有不对之处，欢迎指正。

oss我目前知道的好处有两个:

  1. 静态资源生成链接，静态网站可以进行托管
  2. 用户可以上传图片到这里,从而节约自己服务器的存储空间

oss的简单使用流程

  1. 阿里云的官网注册账号
  2. 点击这里开通对象存储
  3. 新建bucket
  4. js直传的介绍和下载源码
  5. 得到accessKeys
  6. 设置可跨域
  7. 可以直接在控制台上传文件，在右边的设置里就可以看到链接如http://hanfiles.oss-cn-beijing.aliyuncs.com/3.jpg
  8. 打开刚下载的源码，打开upload.js，设置成自己的，比如我的host就是http://hanfiles.oss-cn-beijing.aliyuncs.com，然后在浏览器中打开index.html，点击选择图片 =》 上传图片，然后在控制台里看，就发现图片上传上去了~实际生产情况，一般是前端不公开accessId和key，又后端处理，具体参照官网。

付费情况我没具体细究，但好像看到如果每小时访问量低于万次应该是不收费的，估摸着我的也不会超过十次=。=

使用ossUpload

官方demo，我自己的demo

  • 开通RAM
  • 打开 RAM 控制台
  • 新建一个用户
  • 为用户添加 AliyunSTSAssumeRoleAccess 权限
  • 为用户创建一对 AK, 记录下这个 AK, 控制台不会为你保存
  • 新建一个角色
  • 为角色添加 AliyunOSSFullAccess 权限, 记录下角色 ARN

zip的命令

打包并压缩成文件名为 .zip 的文件zip [-选项] [文件名.zip] [文件名]

参数 demo 含义
无参数 zip new1.zip my1\*.doc 把所有 my1*.doc 的文件打包成一个 new1.zip文件。new1.zip 可以省略扩展名简写成 new1，它会自行默认 .zip 的扩展名。
-d zip -d new1.zip my18.doc 如果当初my18.doc 是压缩在 new1.zip 中的一个文件，现在这条命令把它从压缩文件中抽走并删掉。-d 是删除的意思。
-g zip -g new1.zip my32.doc 如果当初并没有把 my32.doc 压在 new1.zip 里面，现在把它加进去。-g 选项是增加而不要重新产生的意思（grow，长大，这个字母用得比较特别一点）。
-u zip -u new1.zip my2*.doc 如果当初 my2*.doc都已经压缩进 new1.zip 里面了，但是后来 my2*.doc中有几个文件有更改，又不想去找出到底是哪些文件更改了，这条命令请系统自行帮助检查，更改过的才更新到压缩，否则就不必动。-u 是更新的意思。
-r zip -r new2.zip mydir 如果 mydir 是一个目录名称，它会把整个目录的所有文件压缩成一个 new2.zip 文件。-r 选项是压缩子目录的意思。
