vue的列表渲染

发表于 2017-05-17 | 分类于 js |

我们用 v-for 指令根据一组数组的选项列表进行渲染。 v-for 指令需要以 item in items 形式的特殊语法， items 是源数据数组并且 item 是数组元素迭代的别名。

TL;DR

  • 用于数组,index是可选项：v-for="(item,index) in items"
  • 用于对象,key,index是可选项：v-for="(value,key,index) in obj",顺序和Object.keys一致
  • v-for可以用于整数迭代：v-for = "n in 5"
  • v-for和v-if同时使用的情况
  • 数组的变异和重塑
  • 数组的index设值和length截断数组失效，用splice解决
  • 重新显示排序后的数组，用computed和methods

vue的模板语法

发表于 2017-04-01 | 分类于 js |

TL;DR

  • 插值的四种用法：
    • 文本插值
    • html插值v-html="rawHtml"
    • 属性插值v-bind:id="someId"
    • js配合Mustache插值17
  • 指令（Directives）是带有 v- 前缀的特殊属性职责就是当其表达式的值改变时相应地将某些行为应用到 DOM 上。
  • 一些指令能接受一个“参数”，在指令后以冒号指明。v-bind:href="url
  • 修饰符（Modifiers）用于指出一个指令应该以特殊方式绑定。v-on:submit.prevent="onSubmit"，一般用于v-on,v-model
  • 过滤器可以用在两个地方：mustache 插值和 v-bind 表达式。
  • Vue.js 允许你自定义过滤器，可被用作一些常见的文本格式化。
  • 缩写 :
    • v-bind:href=”url -> :href=”url”
    • v-on:click=”doSomething” -> @click=”doSomething”
