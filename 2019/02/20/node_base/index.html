根据现成的promise用法逆推Promise的写法，这边只实现第一个then，没有实现链式写法

  • promise是一个类，参数是一个函数
  • 实例的状态三种，pending fulfilled rejected
  • 实例的值和reason默认是undefined
  • 构造器里面声明了resolve和reject函数,resolve函数执行的前提是状态是pending,执行之后实例的值和状态就变化了，执行then成功的回调
  • 参数函数称为executor，自动执行
  • 实例有存放then成功的回调和存放then失败的回调
  • then在写的时候，通常还是pending状态，所以这里的关键步骤就是将成功的函数放进去，
