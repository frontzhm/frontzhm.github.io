1-冰山模型--找到喜爱的工作

发表于 2019-11-22 | 分类于 career_plan |

什么是冰山模型

冰山模型：全面描述一个人的素质要素。

冰山，从想象中，有一块露出海平面，但部分隐藏在海里。而冰山模型，就是表达这个感觉。

冰山模型分为两大块，30%是知识和技能，70%是能力和价值观、性格特质、动机。前面是看到的部分，后面是海里的部分。

冰山模型干嘛的

可以用冰山模型分析以下：

  • 你和岗位是否匹配
  • 匹配程度
  • 市场薪资

知识和技能

知识是你知道的东西，比如财务知识，一个公式

技能是你能干的事情，比如能开车、能用excel

一个人的知识和技能是相对容易测试出来的，而且可以通过后天习得。

怎么判断是知识和技能不足呢

当工作中每天有大量陌生的内容，觉得信息量很大，感到慌乱和焦虑，那很可能就是知识和技能不足。

解决方案也很简单，因为可以后天习得，所以自己好好学，过一阵子基本就没这问题了，特别容易出现在刚入职的时候。

能力

能力是通用的能力，比如学习能力、思考能力，人际交往能力。

能力不太容易测试出来，需要长期的观察，所以在冰山下面。因为是通用的，所以很容易在各个职业中迁移。

怎么判断是能力不足呢

当工作效率低、沟通效率低，面对复杂的问题无从下手，缺乏成就感，力不从心，很可能就是能力不足了。
解决方法可能就稍微麻烦些，能力的培养周期一般需要好几个月的时间。

价值观、性格、动机

价值观、性格、动机三者在成年以后很难改变，如果工作遇上这些不匹配，很可能意味着你不适合当前岗位

价值观

价值观是你判断事务的标准。如事业和家庭发生冲突的时候，你怎么选择。

怎么判断是不是价值观不匹配呢

工作中常常陷入矛盾和纠结，对所做的事情很难发自内心的认同，很可能就是价值观不匹配。比如公司让你推销一款你觉得不好的产品

性格特质

性格特质是个人的行为偏好。比如你更喜欢独处还是和众人在一起，偏内向或者外向。

怎么判断是不是性格特质不匹配呢

工作中发现工作量不大，但就是心累。很可能就是性格不匹配，比如你喜欢独处，但是工作是销售，总要和不同的人沟通

动机

动机就是做某件事的动力，主要分为三类：

  • 成就动机：喜欢挑战
  • 权力动机：希望影响他人
  • 亲和动机：希望维持好的团队关系

怎么判断是不是动机不匹配呢

工作中发现自己没动力，提不起来劲儿，很可能就是动机不匹配。比如你是是成就动机，喜欢一定的挑战，但是做的工作单一重复

为什么不喜欢一份工作

其实综上，就能大约推断出，如果不喜欢一份工作，是哪方面的原因：

  • 感觉工作内容陌生，信息量大 => 缺乏知识和技能，好好学习，一段时间就好了
  • 感觉工作效率低下，不太会沟通，遇到问题不会解决 => 缺乏相应的能力，好好培养几个月就好了
  • 感觉不认同工作内容，觉得这么做不好 => 价值观不匹配，考虑换。。。
  • 感觉做事提不起来劲儿 => 动机不匹配，考虑换。。。
  • 感觉事不多，但是心好累 => 性格特质不匹配，考虑换。。。

怎么找到分析目标岗位呢

分为4步：

  • 确定分析的岗位。如果没法确定自己的目标岗位，可以去领英等网站搜你的岗位，然后出现很多人，可以看下他们后来去了哪里，这里也能看到未来要去的行业有没有前途
  • 搜索招聘网站上这个岗位，多看几份，提取共性。
  • 按照冰山模型分析这些共性
  • 对比自己和冰山模型分析共性的结果的匹配程度。

分析的多了，你就看到适合自己的职业。

冰山模型还能干嘛

hr会用冰山模型分析你，从而决定你的市场价值即薪资。

方法论和思维模型通常能解决很多问题，掌握一定数量的方法论和思维模型可以解决90%的问题。
冰山模型将一个人的素质全面展开。其当然还能择偶，知道侧重子女教育的哪方面。
所以要努力培养思维模式哇~

实践

分析你是否喜欢当前的工作：

  • 知识和技能大约匹配
  • 能力大约匹配
  • 价值观、性格特质、动机大约匹配

喜欢当前的工作。想要往上一个层级，需要努力在知识和技能上努力
