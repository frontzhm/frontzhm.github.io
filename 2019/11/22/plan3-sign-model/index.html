3-sign模型找到你的天赋

发表于 2019-11-22 | 分类于 career_plan |

本文目标

找到自己的天赋，学习事半功倍。

天赋是啥

天赋是隐藏的能力，让一个人在同样起点下更快速的成长。天赋是催化剂，加快成长，但转化为能力需要刻意练习。

怎么发现自己的天赋

sign模型：

  • 自我效能 self-efficacy：对于某些任务，还没开始就相信自己肯定能做好。比如看到琐碎的任务感觉做不好，但是一个难题的时候就相信自己能做好，就说明你可能在相对单纯的任务更加有天赋
  • 本能 instinct：当你还没开始做这件事的时候，就兴奋、迫不及待的想试试。比如开发课程，就忍不住兴奋，就说明可能在开发课程方面有天赋
  • 成长 growth：学习一个东西的时候觉得时间过得飞快，而且进步的也比别人快点。比如学商业模式一上午一下子就过去了，就说明可能在商业模式上有天赋
  • 满足 needs：即使做这件事很累，但是还是很开心

将满足sign模型的事件提取共性，就说明在这个共性上面有天赋。

我们要做的就说找到天赋，然后刻意练习，转化成能力。
