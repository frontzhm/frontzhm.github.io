命令行开启交互模式

交互模式通常称为“解析器”，常用来实验很短的代码。符合语句，需要手动缩进。
交互模式会显示函数的返回值。

  • 建一个chance.py
  • cd 其所在的文件夹
  • python3
  • import chance
  • chance.coin_flip()
  • exit()终止

注意必须同个文件夹，import的时候没有后缀，调用的时候需要加模块名

命令行运行文件

命令行直接运行文件，不能直接显示返回值，需要通过print。，返回值和输出值有区别

  • 建一个chance.py
  • python3 xxxxx/chance.py

函数没有返回值就是None

字符串

print打印字符串的时候，单引号、双引号。
字符串中有单引号和双引号的话就使用\， 'i\'m hi'。
如果很多换行的话可以print(""" ...xx ...xx ...xxx """)。

表情字符串

字符串和数组:

  • 可以使用len("xx") len([13,2])计算长度，其他类型会报错。
  • 可以使用for循环
  • 可以通过索引拿到各项值，"xx"[0] [12,3][0]。-1表示最后一个字符串，-2表示最后第二个字符串。
  • 可以 123[0:2] [1,2,3][0:2]相当slice，切片操作永远不会导致 IndexError，多的话就是最后一个。如果 word 是一个字符串，word[:] 将是 word 的副本。
  • +的两边要不都是字符串，要不都是数字，不然就会报错

字符串的其他操作

  • 字符串拼接f跟es6类似的操作
  • ==、< 和 > 等比较运算符在字符串上的效果和数字一样。A是65，a是97。
  • 字符串可以转换为数字int('1') float('1.23')，数字可以转化为字符串str(1)

布尔运算

布尔值 True和False
布尔运算 and or not

这些规则的有趣现象是 not (x and y) 和 (not x) or (not y) 一样，同样，not (x or y) 和 (not x) and (not y) 一样。

  • 对于数字，零值被视为 false，所有非零值都被视为 true。
  • 对于字符串，空字符串被视为 false，所有非空字符串都被视为 true。
  • 对于列表，空列表被视为 false，所有非空列表都被视为 true。

序列

字符串和列表是序列
在 Python 中，列表和字符串具有一些共同点。它们都是序列类型 — 它们表示一系列值，而不是单个值。字符串中的项目是字符；列表中的项目可以是任何类型。
