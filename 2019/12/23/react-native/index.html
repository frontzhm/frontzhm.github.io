练习

  • 自定义的组件是Component的子类，必须有render函数，返回一个元素
  • 主要标签View Text Image，View像是div，Text像是span
  • Image必须有source属性，属性值是一个对象里面必须有uri属性，当然还可以设置宽高
  • 组件在用的时候还可以props属性，用法和react大同小异
  • 还可以state，取值的时候this.state.xxx，设置的时候this.setState({isShowingText:true}) this.setState(prevState=>({isShowingText:!prevState.isShowingText}))
  • componentDidMount组件渲染完之后执行的钩子函数
  • state在render的同级
  • 字的颜色之类的style需要引用StyleSheet，然后在Text的标签上style={}
  • 宽高之类的需要在View上，View元素必须有 宽高或者flex 才显示
  • flex默认是colomn的，横向的需要设置style={flex: 1, flexDirection: 'row'}}，justifyContent alignItems
  • 文本输入框TextInput，onChangeText，onSubmitEditing
  • 按钮 Button，title就是显示在按钮上的内容，color可以是按钮颜色，字默认是白色的<Button onPress={() => { alert('You tapped!') }} title="Press Me" />
  • style创建之后可以放在文件末尾，文件里能正常调用
  • 其他能捕获到交互的组件，onPress，onLongPress，TouchableHighlight TouchableOpacity TouchableNativeFeedback TouchableWithoutFeedback
  • 视图能放大缩小滚动，这些就用到scrollView，默认水平竖直都行，可以只设置某一个
  • 列表也有单独的组件FlatList SectionList，flat是每项都一样，section是每项里面加了个头
  • 发请求Fetch，常和FlatList ActivityIndicator在一起用
