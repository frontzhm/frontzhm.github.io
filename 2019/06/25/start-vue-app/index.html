vuecli建立项目，碰到的问题汇总

发表于 2019-06-25 | 分类于 js |

如果需要自定义配置.

打包去掉console.log

注意一般在process.env.NODE_ENV === ‘production’的时候才会压缩，所以如果这个值不是的话不一定会压缩。

参考文档

本地开启伪接口服务，防止测试接口挂掉

  • 新建.env.local，写上NODE_ENV = 'local'
  • package.json配置"local": "vue-cli-service serve --mode local"
  • 根目录写个server.js
  • 相应的在ajax.js那边，配置if (process.env.NODE_ENV === 'local') { return '//localhost:3000' }
  • 然后如果想要开始本地接口，开启两个小窗口，一个执行node server.js，另一个执行npm run local

设置process.env.NODE_ENV

参考文档

  • 根目录新建.env.test，里面写上NODE_ENV = 'test'
  • package.json里面配置"test": "vue-cli-service serve --mode test",，重点是mode为test，命令可以是build
  • 这时候项目里的process.env.NODE_ENV就是test了

配置build的时候js或者css的前缀，默认是/

官方文档

这边的修改，因为服务器那边就是静态资源，需要生成dist的时候，js前缀是./
