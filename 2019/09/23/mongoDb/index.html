mongodb是啥

mongodb 分布式文件存储的开源数据库系统

mongodb将数据存储为一个文档，数据结构由键值对组成，类似于json对象，字段值开源包含其他文档。

『 MongoDB 』 是由 C++ 语言编写的, 基于分布式文件存储的数据库. 属于 NoSql (非关系型数据库) 的一种, 旨在为 WEB 应用提供可扩展的高性能数据存储解决方案.
