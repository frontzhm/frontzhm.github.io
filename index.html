可转债的入门

发表于 2019-11-19 | 分类于 理财 |

可转债全称可转换公司债券，由上市公司发行的，在一定条件下，可转换为公司股票的债券。具有债性和股性的双重属性。

优点：在会买的前提下，可以保证本金，但同时有机会享有牛市带来的收益。低风险高收益。

公司发行可转债需要什么条件呢

  • 上市公司，连续三年盈利且净资产收益率在10%以上。可转债发行之后，资产负债率不超过70%，累计债券余额不超过40%。公司募集资金的投向合法
  • 可转债的发行额不少于人民币1亿
  • 可转债的利率不超过银行同期存款的利率水平

为啥能保本

两个小原因：

  • 目前的没有不还的上司公司
  • 不还的成本远大于还的成本

可转债本质是债券，债券就是欠条，欠条里写明谁借了钱，每年的利息，啥时还钱。可转债就是上司公司向投资者借钱，法律会强制让其到期按欠条还钱。

但注意，不是任意时刻买都能保本，因为你可能高价买了。

  • 可转债的票面面值是固定的100元
  • 可转债的交易价格是随着市场波动的，一般在100左右浮动，所以，只要我们以不高于票面面值100元购买，就肯定会保本。
  • 查看可转债的网站是集思录，将现价这栏从小到大排列，就能低于100的
  • 可转债的到期收益率，等于 你的收益/你买的价格，只要大于0，就能保本
  • 可转债的到期赎回价，到期后上司公司赎回的价格，这个将包含最后一年的利息。

怎么能高收益

拿到高收益的两种方式：

  • 可转债转股后赚取股票上涨的利益
  • 不用转股，直接卖出可转债，赚取可转债的交易差价

可转债的转股价格：购买的时候，合同上写死的价格，固定值。
可转债的正股价格：就是股票的价格，随时会变

棕色，正股价格高于转股价格的时候，投资者就有赚钱的空间了

公司为啥发行可转债

因为这样筹钱的性价比最高。

公司筹集资金有哪些方式呢？

  • 债权融资：相当于打欠条，利息高
  • 股权融资：相当于拉合伙人，收益同享，就是赚了一百万，可能分人家50万
  • 可转债融资，半债性半股性：利息低，如果转股，就不用还钱，所以公司一般希望转股

公司有强制赎回的权利

如果正股价格远高于转股价格，这样投资者的利润就会大大增加，而相对的，上司公司的利益就会减少，所以会有个强制赎回的政策。

强制赎回的条件：连续30个交易日，有15个交易日，正股价都在转股价130%以上

其实一旦到达这个条件，公司就权利执行强制赎回：公告投资者，在某某日要强制赎回了，赎回的话就是普通的低利率。
对于投资者也是利好消息，因为130%了吖，这时候果断转股卖出，千万不要被强制赎回。

公司有下调转股价的权利

下调转股价的条件：连续20(30)个交易日，有10(15)个以上交易日低于转股价格的90%(85% 80%)。
一般会在合同里声明，达到这个条件，上司公司有权利下调转股价，但一般不轻易下调，因为一旦涨回来对上司公司不利，所以上司公司更有可能在快要到期的时候，如果触发这个条件，又不想回售投资者的可转债，就会下调转股价，促使投资者卖掉或者转股，这样就不用还钱

同样，下调转股价对于投资者一样是利好消息。

投资者有回售的权利

一般合同里会写，回售的条件，当达到回售的条件，投资者可以将可转债回售给公司，一般是票面价+利息或者固定的一个值。
但不同的可转债，回售条款也不同。总的来说，对于投资者是一把保护伞。

怎么挑选可转债

挑选可转债主要5要素：

  • 信用评级：AA+以上的比较靠谱，最低的是CCC，一共九个等级
  • 到期收益率：大于0，可保本，当然越大越好。这个属性体现债性
  • 剩余年限：还有多久到期，利益差不多的情况下，当然越小越好
  • 可转债现价：当前购买的价格，当然越低越好，毕竟面值始终是100
  • 溢价率：当前购买这个可转债，立即转换为股票划算的程度，越接近0越划算。这个属性体现股性

挑选大原则

  • 信用评级AA+以上
  • 看下上司公司目前的基本面，以及行业的活跃度，如智能行业正在崛起

稳健者的继续操作：面值小于100且到期税后收益率为正的情况下挑选大的，就肯定可以保本加息

激进派的继续操作：挑选溢价率偏低的，低于5%，没有进入转股期的这栏为灰色。转股期一般是上市6个月之后

想拿到高收益的小技巧

两个小技巧：

  • 预测 上司公司下调转股价
  • 观察快到回售期，但正股价远低于转股价

怎么预测上司公司下调转股价？

  • 打开集思录，按照剩余年限从小到大排列，两年内到期都是备选
  • 选正股价低于转股价，就是转股后卖出就会亏本，显然越低投资者就越可能回售，上司公司为了促使转股，有概率下调
  • 看投资者手里有多少可转债的资金，越大越容易下调
  • 看到期收益率，如果是大于 -2%，可能就不一定参与博弈了，因为博弈失败，就可能亏本。但还是愿意拿出一点代价博弈的
  • 看行业，基本面，负面新闻。财务报表是不是盈利的。

观察快到回售期，但正股价远低于转股价？

  • 密切观察快到期且已经触发回售条件，有的是最后两年，有的是三年，这时候投资者可以回售可债券
  • 如果此时正股远低于转股价，为了避免投资者退货，上市公司也可能会下调转股价，让投资者获利，避免退货。

怎么交易可转债

两种方式：

  • 让基金经理帮你挑选可转债并进行投资，也就是挑选可转债基金，适合上班族
  • 直接交易可转债：开通普通的证券交易账户，就可以在可转债的交易平台交易。

直接交易可转债的两种方式：

  • 可转债的新债申购，俗称打新
  • 已上市的可转债的买卖交易

怎么打新？

  • 先做好功课，看可转债哪一天可申购，因为只有一天可申购
  • 去交易页面，找到新债申购
  • 点击确认申购：中签率特低，一般默认按照顶格数量申购
  • 申购完成，等待结果

怎么交易已上市的可转债？

  • 在广发证券开户后，打开广发易淘金，点击下面的交易
  • 点击买入，输入代码，输入价格，输入数量，点击购买

象限投资法

  • 年到期收益高，溢价率低。就是忒好，可债可股，可遇不可求。
  • 年到期收益高，溢价率高。可债，不一定能股
  • 年到期收益低，溢价率低。可股，不一定能债，最好不长期持有
  • 年到期收益低，溢价率高。别碰了。

什么时候买入卖出

买入：看中债性，到期收益率大于0。看着股性，溢价率小于5%
卖出：

  • 正股价大于转股的130%，可卖出
  • 可转债有7%以上的收益，可卖出
  • 找到历史最高价的肩膀，一旦跌破历史的10%，或者下跌10元，立马卖出

禁区：

  • 不要短线频繁交易
  • 被迫卖出
  • 集中投资

取余运算的使用

发表于 2019-11-14 | 分类于 js |

取余运算，就是拿到余数，比如5➗2=2...1，余数就是1，就是5里面有几个2，剩下的数就是余数1。

取余运算a%b=c，b的取值范围为正整数，a的取值范围是自然数（正整数和0）其有以下的规律：

  • c永远都小于b，且为自然数
  • 若a == 0，则c始终是0，与b无关
  • 若a < b，则c永远都是a，b再大也无关，如 1%6 1%60 1%900，均是1，也就是一旦a<b，那么b是多少都影响不了a
  • 若b==1,，则c始终是0，与a无关
  • 若a是b的整数倍，则c始终是0，
  • 若c是0，a可能是0，也可能a是b的整数倍，也可能b是1

敲黑板！！！正因为c永远都小于b，常用模运算来创建重复动作。

常见方法是将它用于 递增的数字 序列，如b等于2，0%2=0 1%2=1 2%2=0 3%2=1，随着a的变大，取余的值就始终是0和1的循环，所以模数适合在两个（或多个）操作之间切换

比如有2个重复动作，就会n%2，值只会0、1的重复
比如有3个重复动作，就会n%3，值只会0、1、2的重复

发展是三个范畴的统称：生理发展、认知发展、社会化和情感的发展。
三者缺一不可，相互交织，相辅相成。
三个范畴均衡发展才是健康的发展，才是一棵树，不必死磕一个枝丫。
举个例子，2-3岁孩子不能主动和别人一起玩很正常，想解决的话，一般从三个方面着手：

  • 独立感：她确认自己可以能独立做一些事，生活中可以提供一些选择可以让其选择
  • 能力感：她知道自己能力的边界，会做什么不会做什么，生活中鼓励也提供机会让其尝试
  • 家庭氛围：她知道家人就是安全港湾，父母要尽可能提供安全感

因此，带她出去玩，她想怎么玩就怎么玩，不要违背她的意愿，上面三个条件都很棒的情况下，能和别人一起玩的概率就会大大提高。

有些孩子很内向的话，比如被抢了玩具，她本身不把这个当做一回事，父母也就不要太在意，重点还是培养上面三个点，长期以往，情况明显好转。

改变思维模式

发表于 2019-11-12 | 分类于 从心理解孩子 |

直接的语音地址

当孩子的行为出现问题的时候，怎么思考？探究其根源，非表象

很多时候需要反思自己的与孩子的互动，是否尊重她，给她自由的探索空间，让她自己做一些能力范围内的选择。
比如给孩子报了很多兴趣班，但孩子不想上，怎么办？
其实这时候应该想的是，不想上只是表面现象，和孩子从小到大的互动中，父母是怎么互动的可能是主要根源。有没有尊重孩子的兴趣，有没有尊重他自由探索，是她感兴趣的兴趣班，还是妈妈喜欢的兴趣班。

对孩子的期望要符合他们的发展状况，要真心接纳他们

比如上学刚开始的时候磨磨蹭蹭的，其实很正常，大脑的执行功能没有成熟，此时经常和他们讨论时间表，把时间表贴在墙上，虽然催但不会人身攻击，久而久之，到了三年级，就好多了

比如作业管理，刚开始的时候拿回来一张纸，做完了，然后就不知道作业纸去哪里了，此时和他们说，要学会自己对作业负责，并且辅助放个塑料框，写完放进去，久而久之，到了三年级，就好多了

比如书包管理，写好清单，要放的东西，前天晚上就收拾好。慢慢的养成习惯了就好了。

比如作业忘了带，他们自己知道就好了，不必苛责。慢慢给他们耐心，他们会慢慢长大的。忘记带的次数会越来越少的~

讲道理归讲道理，但一定优先考虑孩子的感受

违背孩子感受的讲道理，会让孩子不舒服，愤怒甚至逆反。很多时候宽容理解，从而让孩子更加明白自己的责任感。
想想你的态度、语言以及传递给孩子的价值观是什么

比如忘记带作业第三次了，孩子并不是故意的，他本人也已经不开心了，此时应该抱抱他，跟他说虽然没带造成一些不开心，但也不要沮丧，以后尽量记得就好了，而不是一味苛责他。

发展是连续的，不要妄图让孩子在短时间内脱胎换骨，慢慢培养他们的责任心，能力，给他们耐心让他们成长成熟。切忌不能操纵他们，一人专制。
