正则表达式

发表于 2018-02-09 | 分类于 js |

正则开始比较发怵的一个东西，但突然间想玩好。搜了下相关文章，觉得快速攻陷正则表达式这篇写的能有耐心看下去。。。。

正则干嘛的

首先，我们会经常用自然语言描述一类字符串，如 以 “010 开头的电话号码”、“含有 hello 的字符串”、“IP地址”等等。

正则表达式就是一种描述性的语言，用来概括一类字符串 (或者说一个字符串集合)。 如 正则表达式/hello/就用来概括“含有 hello 的字符串”

上面的定义请读三遍。简言之，用一个表达式, 去描述一类字符串(或者说, 一个字符串集合)。

最基本的正则表达式

  1. 一个字符串就是一个表达式。比如 aaa，就是一个正则表达式，它描述了一个字符串集合, 这个字符串集合里面只有 aaa 这一个元素。
  2. 两个表达式可以串起来。比如 aaa和bbb可以串成aaabbb
  3. 两个表达式用|连起来，表示这两个表达式的并集。比如aaa|bbb表示aaa和bbb，也就是 aaa和bbb各自集合的并集（别跟我说你不知道什么是并集）

拓展

最基本的正则表达式只能表示有限的字符串集合，也就是你用|一直连接。。。。。。这是不行滴，不够用滴。于是加上一些新的记号来表示无限的集合。

  1. \*: 一个字符串 X 可以加上一个 *， 用来描述任意多个原来 X 描述的字符串拼起来的字符串。定义看的好定义。。。比如 a* 这个正则表达式, 我们知道 a 描述了一个字符串集合，这个集合里面只有a这一个元素 ，加上*之后，就表示这个集合里面有 a、aa、aaa、aaaa等等也就是任意数量的a拼起来的集合。其实还有还有特殊情况，因为是任意数量，所以可以是0个a也就是空字符串也在这个集合里。
  2. ()：整体。先说 a|b*表示集合 a、’’、b、bb、bbb等等，如果引入括号，(a|b)*，那么 a|b就变成一个整体，描述了a或者b，表示只有a和b组成的字符串
