注意户外时间和睡觉

不爱喝奶怎么办

这个阶段宝宝容易出现阶段性厌奶,一段时间爱喝,一段时间不爱喝…
可以试试: 

1. 换容器: 吸嘴杯,敞口杯,鸭嘴杯. 有些杯身上面有宝宝喜欢的卡通图案可能也能增加宝宝喝奶的兴趣
2. 游戏喝奶: 
    1. 宝宝喝奶的时候大人也拿着杯子喝牛奶,并且相互碰杯,这个阶段宝宝对节奏感很敏感,有时候这个方法就挺好用的
    1. 用两个吸管,假装和宝宝一起抢奶喝~
    1. 喝一口奶就击掌下等等, 开发脑洞的想吧~
3. 奶做到食物里面: 用奶炖蛋,用奶和面,用奶煮粥等等虽然烹饪过程中损失部分营养,但大部分营养都还在的
4. 补充其他奶制品: 酸奶,奶酪(酸奶拌水果 奶酪焖饭),这里的无糖酸奶和原汁奶酪

10月的辅食制作公式

辅食颗粒

比起前面的,这次更加颗粒,剁碎生肉,剁碎蔬菜,切下玉米颗粒,切分颗粒

软饭

米和水的比例是1:3 或者 1:4

软饭的一般步骤1: 

煮熟米饭 -> 加入配菜焖软或者炒在一起

软饭的一般步骤2: 

米和配菜一起煮熟(比如胡萝卜,黄瓜和土豆,肉)

原料: 主食 + 液体 + 配料 

1. 粮食主食: 大米,小米,玉米,高粱米,燕麦,红豆,绿豆,红薯,紫薯
2. 液体: 水 奶 高汤
3. 配料: 根茎类可以一起炖, 莴笋,胡萝卜,土豆,肉的话天气热可以煮熟之后跟饭炒熟就行

比如: 胡萝卜土豆干贝焖软饭

原料: 主食(大米) + 液体(水) + 配料(胡萝卜 土豆 干贝)

比如: 番茄西蓝花鸡蛋炒饭

原料: 主食(大米 小米) + 液体(水) + 配料(番茄 西蓝花 鸡蛋)

意面

一般步骤: 

意大利面煮熟 - 加入配菜炒匀

原料: 面(意大利面) + 酱料(青酱 红酱 白酱) + 配料

白酱: 面粉牛奶黄油
红酱: 番茄肉末
绿酱: 菠菜或西蓝花

多做一些酱,冰冻起来.

配料有很多: 如青菜 胡萝卜 蘑菇 虾 鸡蛋等

比如: 胡萝卜番茄肉意面 

原料: 面(卡通意面) + 酱料(番茄 肉) + 配料(胡萝卜)

比如: 牛肉洋葱白酱意面

原料: 面(螺旋意面) + 酱料(白酱) + 配料(牛肉 洋葱 西蓝花)

包子

一般步骤:
擀皮 - 包馅皮 - 蒸熟

原料:
粉(面粉 可以加点全麦粉 玉米面 米粉 坚果粉之类) - 液体 (安琪干酵母 奶 水 ) - 馅料(红薯 紫薯 猪肉青菜 香菇 胡萝卜)
香菇 胡萝卜等不容易熟的可以提前炒熟

比如: 香菇猪肉玉米包子

原料: 粉(面粉 玉米面) + 液体(水) + 馅料(猪肉 香菇)

比如: 奶香紫薯包子

原料: 粉(面粉) + 液体(牛奶) + 馅料(紫薯泥)

汤

小月龄基本不太缺水.
大月龄干稀搭配.

奶糊

玉米奶糊
酸奶+燕麦+芝麻
牛油果牛奶

小炒

加植物油.
番茄炒豆腐
干贝鸡蛋白菜
可以观察宝宝喜欢的食物

蒸炖

生的食物都放一个锅里蒸,这样很方面
炖也是一样的

11月龄制作公式

这个月龄之后基本可以接触大人食物,基本适应所有食物

12月龄之后的辅食

1岁之后,食物就是主要了.也不是婴儿,而是幼儿了.

食物搭配: 大概框架 => 渐渐细化

基本需要的原料: 蔬菜 + 谷类 + 蛋肉 + 水果 + 奶(4 + 3 + 3 + 2 + 1)

步骤: 

1. 列出采购清单
2. 提前处理工作:预约 库存 预处理
3. 食物搭配注意点

食物搭配注意点(周/月): 

1. 未加盐: 每周1-2次含碘食物
2. 贫血: 每周1-2次肝脏
3. 每周2-3次水产
4. 每周吃点豆制品
5. 粗粮占1/5-1/4
6. 吃较丰富的绿叶蔬菜

12月龄辅食和奶

进食时间

  1. 三餐与家人同步,基本能吃一样食材
  2. 尽早培养吃早餐的习惯 (7点起来先吃早餐 9.30喝奶加餐 12.0午餐)
  3. 晨奶位置调整
  4. 戒夜奶 (温和渐进[入睡到醒来]] 简单粗暴[妈妈不出现 白开水代替])

正餐和奶


并觉期的处理

上午觉取消了,上午清醒时间变长,7.0-12.0 ,吃饭就会很困.
解决办法: 午餐分两顿,睡前半餐,睡后半餐
适合半餐的食物: 吐司 包子 饺子 蒸玉米 蒸红薯 鸡蛋饼 寿司
一岁半到两岁的时候,并觉期过了就可以正常了.
